🎯 الخطوات السريعة (6 خطوات فقط):

1. Build على جهازك المحلي:
   .\DEPLOY_PRODUCTION.ps1 -Action build

2. جهز السيرفر (Ubuntu/Debian):
   curl -fsSL https://get.docker.com | sh
   sudo apt install docker-compose

3. انسخ المشروع على السيرفر:
   git clone https://github.com/your-username/care-management-system.git

4. أنشئ ملف .env:
   cp .env.example .env
   # ثم عدّل البيانات الحساسة

5. شغّل الخدمات:
   docker-compose -f docker-compose.prod.yml up -d

6. تحقق من الوصول:
   http://server-ip:3000  (React App)
   http://server-ip:6060/actuator/health  (API Gateway)

🔧 الأوامر الأساسية:

# بناء
docker-compose build

# تشغيل
docker-compose -f docker-compose.prod.yml up -d

# إيقاف
docker-compose down

# Logs
docker-compose logs -f

# Health check
curl http://localhost:6060/actuator/health

# Database
docker exec -it care-postgres psql -U postgres -d cms_db
